# 🎵 Music Player Web App

## 📌 Project Description
A fully functional Music Player built using HTML, CSS, and JavaScript.  
It allows users to play, pause, skip songs, control volume, and view song details.

## 🚀 Features
- Play / Pause functionality
- Next & Previous buttons
- Song title and artist display
- Progress bar with seek control
- Volume control
- Playlist support
- Autoplay next song

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## ▶ How to Run
1. Download or clone the repository
2. Open `musicplayer.html` in your browser
3. Add your own MP3 files in the project folder
